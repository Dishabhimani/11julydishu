Que-2 How to Apply CSS Using JQuery, How to Add Class and Remove Class inJquery, JQuery Animation?

Apply CSS using jQuery:-

You can apply css using css() method.

Example:-

$(document).ready(function(){
$('selector').css('property', 'value');
});

Add and Remove class in jQuery:-

You can add or remove class using jQuery's addClass() and removeClass() methods.

Example:-

$(document).ready(function(){
$('selector').addClass('classname');
$('selector').removeClass('classname');
});

Animation using jQuery:-

You can apply animations in jQuery using this methods animate(), fadeIn(), fadeOut(), slideDown(), slideUp(), etc.

Example:-

$(document).ready(function(){
$('selector').animate({
property: 'value'
}, duration);
});
