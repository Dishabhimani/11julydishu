Que-56 how many type of JS Event? How to use it ?

JavaScript events come in several types, categorized based on their origin or source. Here are the main types of JavaScript events:

  1. Mouse Events: These events occur when a user interacts with the mouse. Examples include click, mouseover, mouseout, etc.
  2. Keyboard Events: These events occur when the user interacts with the keyboard. Examples include keydown, keyup, keypress.
  3. Form Events: These events are related to HTML form elements. Examples include submit, reset, change, focus, blur.
  4. Document/Window Events: These events are related to the loading and unloading of the document or the browser window. Examples include load, resize, scroll, unload.
  5. Event Delegation Events: These events allow handling events on multiple elements using event delegation techniques. Examples include delegate, undelegate.
