Que-57 What is Bom vs Dom in JS?

DOM (Document Object Model):

  • The DOM represents the structure of an HTML or XML document as a hierarchical tree of objects. Each node in the tree represents a part of the document, such as elements, attributes, and text.
  • It provides a structured representation of the document's content and allows scripts to dynamically access and manipulate the document's elements and attributes.
  • Example operations include selecting elements, modifying their content or attributes, adding or removing elements, and responding to events.

    • BOM (Browser Object Model):

  • The BOM represents everything else in the browser environment aside from the document itself. It includes objects such as the window object, which represents the browser window that contains the document.
  • It provides objects and methods to interact with the browser environment, such as navigating to URLs (window.location), managing history (window.history), handling dialogs (window.alert, window.confirm, window.prompt), and timing events (window.setTimeout, window.setInterval).
