Que-15 What is the result of the expression (5 > 3 && 2 < 4)?

The expression (5 > 3 && 2 < 4) involves two comparisons and a logical AND operator:

  • 5 > 3 evaluates to true because 5 is greater than 3.
  • 2 < 4 evaluates to true because 2 is less than 4.
    • The logical AND operator (&&) returns true if both operands are true. Since both 5 > 3 and 2 < 4 are true,so the result of the expression (5 > 3 && 2 < 4) is True.
