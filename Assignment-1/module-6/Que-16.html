Que-16 What is the result of the expression (true && 1 && "hello")?

For the expression (true && 1 && "hello"):

  • true is a boolean value and is considered truthy.
  • 1 is a number and is also considered truthy.
  • "hello" is a string and is also considered truthy.
    • The result of the expression (true && 1 && "hello") is hello.
