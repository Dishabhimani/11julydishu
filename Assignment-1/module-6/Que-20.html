Que-20 What is the difference between && and || in JavaScript?

In JavaScript, && (logical AND) and || (logical OR) are logical operators used to combine multiple boolean expressions or values.

Logical AND (&&)

The && operator returns true only if both operands are true. If any operand is false, the result is false.

Syntax:

expr1 && expr2

  • If expr1 is false, expr1 is returned.
  • If expr1 is true, expr2 is evaluated and its value is returned.

    • Example:

    Logical OR (||)

    The || operator returns true if at least one of the operands is true. If both operands are false, the result is false.

    Syntax:

    expr1 || expr2

  • If expr1 is true, expr1 is returned.
  • If expr1 is false, expr2 is evaluated and its value is returned.

    • Example:

    Key Difference:

    1.Evaluation Logic:

  • && returns true only if both operands are true.
  • || returns true if at least one operand is true.

    • 2.Short-Circuit Evaluation:

  • && stops evaluating and returns the first false operand.
  • || stops evaluating and returns the first true operand.

    • 3.Return Value:

  • && returns the last evaluated value if all are true, otherwise the first false value.
  • || returns the first true value if any, otherwise the last evaluated value if all are false.
