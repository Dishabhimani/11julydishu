Que-3 Define a Data Types in js?

JS have many data types like:

  • String
  • Number
  • Bigint
  • Boolean
  • Undefined
  • Null
  • Symbol
  • Object

    • String:

    Used to store multiple text.

    It is also created using single quote.

    Number:

    It contain numeric values.

    Bigint:

    It is used to store big integer values that are too big to be represented by a normal JavaScript Number.

    Boolean:

    It is represent two values: True OR False

    Undefine:

    It is indicates that a variable has not been assigned a value, or not declared at all.

    Null:

    It is represents no value or no object

    Symbol:

    It represents a unique identifier.

    Object:

    The object data type can contain both built-in objects, and user defined objects.
